# walsoc - Write Any Language SOurce Code

Write source code in any language on a common basis, php-like.

## Working on

- Python
- Ruby

## Coming soon to all theatres

- PHP implementation
- Java implementation
- Proper documentation

## Install

  python3 setup.py install

## Uninstall

  pip uninstall walsoc
